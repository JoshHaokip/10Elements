# :running: Running the app

Clone this repo:

```
$ git clone https://github.com/JoshHaokip/10Elements
```

Install all dependancies:

```
$ npm install
```

Start the server:

```
$ npm start
```

And open the app in your browser at <http://localhost:3000/>.
